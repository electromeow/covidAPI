The COVID-19 JSON API
that directly getting data from WHO

Without need of a token or API key,
without a ratelimit!

Endpoints:

  • GET /global

    Sends the global stats.

  • GET /country/:country

    Parameter country: 2-character country code(e.g. us, in, br) or country name

    Sends the stats for that country.

  • GET /all

    Sends all the data.
